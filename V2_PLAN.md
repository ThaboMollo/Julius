# JULIUS V2 — SIMPLIFIED BUDGETING PLAN

## 🎯 Objective
Reduce onboarding friction to near-zero by:
- Auto-generating a working budget
- Allowing immediate usage without setup
- Gradually revealing complexity

---

## 🧱 1. CORE STRUCTURE

### Default Budget Groups (Auto-created)
- Needs
- Wants
- Savings
- Liabilities

Rules:
- Created automatically on first launch
- Editable later
- Cannot be deleted initially

---

## 📦 Default Categories

### Needs
- Rent
- Groceries
- Transport
- Utilities
- Insurance

### Wants
- Entertainment
- Subscriptions
- Eating Out
- Shopping

### Savings
- Emergency Fund
- Investments

### Liabilities
- Credit Card
- Personal Loan
- Car Payment
- Debt Repayment

Rules:
- No category creation required upfront
- Editable later

---

## ⚡ 2. FIRST-TIME EXPERIENCE

On app open:
- Auto-create current month budget
- Auto-create groups & categories
- Planned amounts = 0

### First Screen
- Money left (default 0)
- Prompt to add income
- Quick Add Expense button

---

## 💸 3. TRANSACTION-FIRST UX

Primary action:
[ + Add Expense ]

Flow:
1. Enter amount
2. Select category
3. Save

No setup required.

---

## 🧠 4. SAFE TO SPEND

Formula:
Safe to spend = Income - (Needs + Liabilities + Savings)

Display:
“You can spend RXXX safely”

Behavior:
- Updates in real time
- Turns red when negative

---

## 💀 5. EXPENSES THAT SHOULD DIE

Detect:
- Subscriptions
- Repeated expenses
- High spend categories

Display:
“You’re wasting RXXXX/month”

Actions:
- Keep
- Cut
- Reassign

---

## 📅 6. COMMITMENTS

Merge:
- Bills
- Debts
- Subscriptions

Fields:
- Name
- Amount
- Due date
- Type

Display:
Upcoming payments list

---

## 🔄 7. RECURRING

Simple toggle:
[ ] Repeat monthly

Auto-creates next month entries.

---

## ☁️ 8. AUTH + SYNC

Rules:
- No login required initially
- Prompt later to save data

After login:
- Silent upload of local data
- Bidirectional sync
- No user interaction

---

## 📊 9. HOME SCREEN

Sections:
1. Safe to spend
2. Waste detection
3. Upcoming commitments
4. Recent transactions
5. Add expense button

---

## 🧩 10. SIMPLIFICATIONS

Remove:
- Manual setup flows
- Template setup
- Empty states

Delay:
- Advanced reports
- Bank uploads
- Multi-account

---

## 🧪 11. SUCCESS METRICS

- Time to first transaction < 10 seconds
- Time to first income < 30 seconds
- High Day 1 retention

---

## 🧠 FINAL POSITIONING

Julius = Instant financial awareness + control

Not a system to configure.
